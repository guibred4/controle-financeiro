# CashQuest - Controle Financeiro Inteligente

Um SaaS moderno para gestão financeira pessoal e familiar, com foco em simplicidade, automação e insights visuais.

## Funcionalidades
- Adição de receitas e despesas (incluindo recorrentes mensais)
- Categorias personalizáveis
- Relatórios e gráficos interativos
- Recuperação de senha por email
- Controle de saldo em tempo real
- Suporte a múltiplos usuários por grupo

## Tecnologias
- Frontend: Streamlit
- Backend: Supabase (PostgreSQL)
- Gráficos: Plotly

## Deploy
Acesse a aplicação em: https://controle-financeiro-rlj3rpwgehdfw8tckwsvok.streamlit.app/

## Como Usar
1. Faça login/cadastro.
2. Adicione receitas e despesas.
3. Visualize resumos e gráficos na aba "Resumo / Gráficos".

## Desenvolvimento
Para rodar localmente:
```bash
pip install -r requirements.txt
streamlit run app.py
```
